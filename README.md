# 🎓 Student Performance Analyzer using NumPy

A menu-driven Python project that analyzes the Student Performance Dataset using **NumPy only**. This project demonstrates data loading, filtering, searching, sorting, statistical analysis, and report generation without using Pandas.

---

## 📌 Project Description

This project is built to practice and strengthen NumPy concepts by performing real-world data analysis on a student performance dataset.

The application is completely menu-driven, allowing users to choose different operations interactively.

---

## 🚀 Features

### 📂 Dataset Operations
- Load dataset from CSV
- Display dataset shape
- Display dataset size
- Show first N records
- Show last N records
- Display any student record
- Display column names
- Dataset summary

### 🎓 Student Analysis
- Search student by row number
- Find topper
- Find lowest scorer
- Display passed students
- Display failed students
- Calculate total marks (G1 + G2 + G3)
- Calculate average marks
- Find highest and lowest marks


### 📊 Statistical Operations
- Sum
- Mean
- Maximum
- Minimum
- Total students
- Total features

### 🛡 Error Handling
- Invalid row number handling
- Invalid user input handling
- Index out of range handling

3. Run the project

